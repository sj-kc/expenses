# Expenses app.

* React.
* Redux.
* React-Router.
* Webpack.
* Babel.
* Firebase.
* Authentication.
* ES6/ES7.
* Sass.
* Heroku.

### Clone the repo

```bash
git clone https://github.com/sungkimcastro/expenses.git
```

### Install dependencies

```bash
yarn install
```

### Start a local server to preview the site

```bash
yarn dev-server
```
